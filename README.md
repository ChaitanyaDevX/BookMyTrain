# 🚆 BookMyTrain

A Java console-based train ticket booking application inspired by IRCTC.

## Features
- 👤 User Registration & Login
- 🔍 Search Trains by Source & Destination
- 🎟️ Book Tickets
- 📋 View Booked Tickets
- ❌ Cancel Tickets
- 🚂 View All Available Trains

## Tech Stack
- Java (Core)
- Collections Framework (HashMap, List)

## How to Run
1. Clone the repository
```bash
   git clone https://github.com/ChaitanyaDevX/BookMyTrain.git
```
2. Open in IntelliJ IDEA or any Java IDE
3. Run `IRCTCAPP.java` as the main class

## Project Structure
```
src/
├── IRCTCAPP.java        # Main app & menu flow
├── UserService.java     # Register, login, logout
├── BookingService.java  # Train search & ticket booking
├── User.java            # User model
├── Train.java           # Train model
└── Ticket.java          # Ticket model
```
